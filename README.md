## Spark Tutorial

### 主要教學內容
- 巨資資料生態系統
	- 巨量資料簡介
	- Hadoop 分散式運算框架
	- Spark 分散式運算框架
	- Linux 基本指令操作
- Spark SQL 結構化資料處理應用
	- SparkSQL
	- DataFrame
	- R
	- SparkR
	- Java
	- MySQL
	- Hive
- Spark 技術實作
	- Spark RDD 核心技術
- Spark Streaming 技術實作
	- DStream
	- Socket Streaming
	- Spark Streaming
